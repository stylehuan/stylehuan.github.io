简单了解下，sass的语法和标准的css语法差不多，只是文件扩展名以.sass结尾。

变量

可以在css预编译器中声明变量。然后在样式文件中使用这些变量。这些变量可以保存任何值。颜色、数值、文本等。

需要注意的是

  • sass声明的变量必须是$开头的。从编程习惯来讲，变量名要语义化。

  • 如果变量需要嵌套再字符串中。就必须写在#{}之中。
